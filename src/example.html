API Documentation

Integrate SmoothScroller into your project with just a few lines of code.

Basic Initialization

// Import the library (ES Module)
import SmoothScroller from 'smooth-scroller';

// Initialize with default options
const scroller = new SmoothScroller({
    element: '#content',      // Element to apply smooth scrolling
    smoothness: 0.95,         // Smoothness factor (0.8-0.98)
    scrollToDuration: 1000,   // Duration of programmatic scrolls
    scrollToEasing: 'easeInOutCubic' // Default easing function
});

Programmatic Control

// Scroll to specific position
scroller.scrollTo(500); // Scroll to 500px from top

// Scroll to element
scroller.scrollToElement('#section-2', {
    offset: 100,       // Additional offset from top
    duration: 1500,    // Custom duration for this scroll
    easing: 'easeOutBack' // Custom easing for this scroll
}).then(() => {
    // Callback when scroll completes
    console.log('Scroll finished!');
});

// Get current scroll position
const position = scroller.getScrollPosition();

// Toggle smooth scrolling
scroller.toggle(true); // Enable/disable smooth scrolling

Configuration Options

Customize SmoothScroller to perfectly match your application's needs.

const scroller = new SmoothScroller({
    // Required: Element to apply smooth scrolling to
    element: '#scrollContainer',

    // Smoothness configuration (0.8-0.98 recommended)
    smoothness: 0.95,       // How smooth the scrolling feels
    minMovement: 0.5,       // Minimum pixels to move per frame
    maxDeltaTime: 2,        // Maximum time delta multiplier

    // Scroll-to behavior
    scrollToDuration: 1000, // Duration of programmatic scrolls
    scrollToEasing: 'easeInOutCubic', // Default easing function
    autoScrollOffset: 0,    // Automatic offset for scroll-to elements

    // Device handling
    useNativeForTouch: true, // Fallback to native scroll on touch devices
    respectReducedMotion: true, // Respect prefers-reduced-motion

    // Debugging
    debug: false,           // Enable debug logging

    // Event hooks
    onInit: () => {},       // Called after initialization
    onDestroy: () => {},    // Called when instance is destroyed
    onScrollStart: () => {}, // Called when scroll starts
    onScrollEnd: () => {}   // Called when scroll ends
});

Available Easing Functions

// Built-in easing functions:
'easeInOutCubic' // Default - smooth acceleration/deceleration
'easeOutQuint'   // Stronger deceleration
'easeOutBack'    // Slight overshoot at the end
'easeOutElastic' // Elastic bounce effect
'easeInOutQuad'  // Subtle quadratic easing
'easeOutExpo'    // Exponential deceleration

// You can also provide custom easing functions:
function customEasing(t) {
    return t < 0.5 ? 2 * t * t : 1 - Math.pow(-2 * t + 2, 2) / 2;
}

new SmoothScroller({
    scrollToEasing: customEasing
});
